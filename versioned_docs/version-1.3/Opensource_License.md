- [A-GPL 3.0](<https://tldrlegal.com/license/gnu-lesser-general-public-license-v3-(lgpl-3)>) for Stand alone app
- [L-GPL 3.0](<https://tldrlegal.com/license/gnu-general-public-license-v3-(gpl-3)>) for Reusable libraries
